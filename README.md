# Data Analysis & EDA Projects

This repository contains multiple **Exploratory Data Analysis (EDA)** and **Data Cleaning projects** using **Python, SQL, and datasets from different domains**.
The goal of these projects is to analyze datasets, clean data, visualize patterns, and extract useful insights.

---

## 📂 Project Files

### 📊 Python EDA Projects (Jupyter Notebooks)

* **EDA-Google playstore data.ipynb**
  Exploratory Data Analysis of the Google Play Store dataset including app ratings, installs, categories, and trends.

* **EDA-HR Analytics.ipynb**
  Analysis of HR dataset to understand employee performance, attrition patterns, and workforce insights.

* **EDA-Travel.ipynb**
  Data exploration of travel dataset to analyze customer behavior and travel patterns.

* **EDA-Amazondata.ipynb**
  Analysis of Amazon sales dataset to understand sales trends, product categories, and performance.

* **EDA-Chronic data.ipynb**
  Analysis of chronic disease dataset to study health-related patterns and statistics.

---

### 🗄️ SQL Projects

* **Data Cleaning Project.sql**
  SQL queries used for cleaning raw datasets including handling null values, duplicates, and formatting.

* **EDA Project.sql**
  SQL queries used to perform data exploration and analysis directly in the database.
